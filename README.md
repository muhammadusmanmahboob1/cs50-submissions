# cs50-submissions
This repository contains the weekly problem sets completed and submitted during Harvard University's CS50x course (Introduction to Computer Science). 

---

# Course Overview

The repository is structured by weeks to demonstrate the submissions of the probelm sets given in the entire course and also the capstone project.

* **`Week 0: Scratch`** — Initial problem-solving and foundational logic via visual programming.
* **`Week 1: C`** — Syntax, data types, logic loops, and terminal-based programming.
* **`Week 2: Arrays`** — Compiling, debugging, cryptography, and data layout.
* **`Week 3: Algorithms`** — Searching and sorting methodology.
* **`Week 4: Memory`** — Pointers, hexadecimals, and custom file manipulation.
* **`Week 5: Data Structures`** — Stacks, queues, linked lists, and hash tables.
* **`Week 6: Python`** — High-level scripting, automation, and syntax translation.
* **`Week 7: SQL`** — Relational database design and data querying.
* **`Week 8: HTML, CSS, JavaScript`** — Front-end UI design and client-side scripting.
* **`Week 9: Flask`** — Back-end web frameworks and dynamic server routing.
* **`Week 10: Final Project`** — Independent software design implementation.
